## 说明

本浏览器扩展程序旨在为日漫爱好者提供更好的阅读体验。

覆盖原有的网页，提供更舒适的阅读环境，并且支持预加载图片和分页修复。

## 特性

- 目前仅支持拷贝漫画
- 预加载图片
- 分页修复
- 键盘快捷键导航（方向键、A/D 翻页，空格键显示菜单）
- 双页模式（宽屏显示两页）
- 章节跳转（上一话/下一话/返回目录）

## 使用方法

1. 安装扩展程序
2. 访问支持的漫画网站
3. 访问漫画页时自动加载阅读器UI

## 技术栈

- TypeScript + Vue 3
- Vite + Sass
- WebExtension API (Manifest V3)
